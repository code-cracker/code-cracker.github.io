# Cause

Catch statements with no Exception as an argument is not recommended. Consider adding an Exception class to the catch statement. 

# Example

{% highlight csharp %}
try
{
    // do something
}
catch
{
    int x = 0;
}
{% endhighlight %}

# Code fix

A code fix will be presented to you that will transform the code:

{% highlight csharp %}
try
{
    // do something
}
catch (Exception ex)
{
    int x = 0;
}
{% endhighlight %}
