# Cause

If-else statements can be changed to ternary operators.

# Example

{% highlight csharp %}
var something = true;
if (something)
    return 1;
else
    return 2;
{% endhighlight %}

# Code fix

A code fix will be presented to you that will transform the code:

{% highlight csharp %}
var something = true;
return something ? 1 : 2;
{% endhighlight %}
